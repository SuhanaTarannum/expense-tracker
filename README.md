# Expense Tracker Pro

A modern web-based **Expense Tracker** that helps users manage their finances efficiently.  
Track income and expenses, set monthly budgets, view monthly analysis, and visualize spending with charts. Features both **light and dark themes** for a sleek experience.

---

## Features

- User authentication (Register & Login)
- Add, edit, and delete transactions
- Monthly budget tracking
- Monthly expense analysis
- Pie chart & bar chart visualization
- Light and dark theme toggle
- Responsive design for desktop and mobile

---

## How to Use

1.Clone the repository or download the ZIP:
2.Open index.html in your browser
3.Register a new account
4.Login to start tracking your expenses
5.Add monthly budgets, expenses, and see analysis in charts
